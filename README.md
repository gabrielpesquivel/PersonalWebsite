## Features

### Interactive Components
- **Animated Loading Screen** - Eye-catching typewriter effect welcome animation
- **Dynamic Experience Carousel** - 3D carousel showcasing professional experience at NextCoin, EY, Sunswift, and Mercury
- **Interactive Skills Display** - Hover-responsive pill tags organized by technology category
- **Blog with Modal View** - Personal tech blog with smooth modal popups for full post reading
- **Mobile Responsive** - Fully optimized for mobile devices with hamburger menu navigation

### Technical Highlights
- Built with **React** and **React Router**
- Smooth scroll navigation using **react-scroll**
- 3D carousel animations with **react-spring-3d-carousel**
- Motion animations using **motion** library
- Modern CSS with responsive design and smooth transitions
- Mobile-first approach with sleek drawer navigation

## Responsive Design

The website is fully responsive with breakpoints at:
- **Mobile:** < 768px - Compact layout with hamburger menu
- **Tablet:** 768px - 1024px - Optimized grid layouts
- **Desktop:** > 1024px - Full-width experience
